# Neural-Networks---Deep-Learning-1
Deep Learning assignment on image classification.

The end goal is to identify pneumonia from X-ray images of the chest cavity. The dataset of choice is PneumoniaMNIST. For that we try using: a fine-tuned CNN we built ourselves, the pretrained ResNet-18 (transfer learning) and the DeiT-Tiny Patch 16-224 Vision Transformer. In the last two cases we train and modify the classifier head as to fit it for binary classification. 

For more details you can inspect the Report.
